### 2. コラム構造を導入したED法とは？（初学者向け導入）
#### 2.1 オリジナルED法の紹介
- 金子勇氏による1999年の発案
- 生物学的に妥当な学習アルゴリズム
- 従来の機械学習との3つの違い
  * 興奮性・抑制性ニューロンペア
  * アミン拡散学習
  * 誤差逆伝播を使わない
#### 2.2 オリジナルED法の課題
- 二値分類には強い
- 多クラス分類は不得手
#### 2.3 解決策：コラム構造の導入
- 大脳皮質のコラム構造からの着想
- 各クラスに対応する専用ニューロン群
- 1つの重み空間で多クラス分類を実現
#### 2.4 本実装の位置付け
- オリジナルED法の理論を100%保持
- コラム構造で多クラス分類を可能に
- 生物学的妥当性を維持
