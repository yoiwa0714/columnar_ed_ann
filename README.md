## 概要

**コラムED法**は、金子勇氏が考案した誤差拡散学習法（ED法、以下オリジナルED法）に、大脳皮質のコラム構造を導入することによってオリジナルED法を拡張したニューラルネットワーク実装です。

コラムED法では**微分の連鎖律を用いた誤差逆伝播法（Backpropagation）を一切使用せず**、生物学的に妥当なアミン拡散機構によって学習を行っていますが、MNIST手書き数字認識で **97.36%** のテスト精度を達成しています（6層構成、訓練50,000サンプル）。

本リポジトリでは以下の2つの実装を提供しています：

| 実装 | ファイル | 用途 |
|------|---------|------|
| **メイン版** | `columnar_ed_ann.py` | 最小限の引数で高精度を実現。本ドキュメントの対象 |
| 実験版 | `columnar_ed_ann_experiment.py` | メイン版よりも多くのパラメータを指定可能。詳細は [README_experiment.md](README_experiment.md) 参照 |

**メイン版**は、**`modules/` ディレクトリのモジュールだけを使用する自己完結型の実装**です。`columnar_ed_ann.py` と `modules/` だけで動作するため、実験版のコードを参照せずにED法とコラム構造の仕組みを理解できます。

## 特徴

### 1. 微分の連鎖律を用いた誤差逆伝播法を使用しない

従来のニューラルネットワークでは「微分の連鎖律を用いた誤差逆伝播法」で重みを更新しますが、本実装ではこれを一切使用しません。代わりに、脳の神経伝達物質（アミン）の拡散をモデル化した機構で学習を行います。

### 2. 大脳皮質のコラム構造

大脳皮質の視覚野に見られるコラム構造を導入し、一部のニューロンを特定のクラスに割り当てます。これにより、オリジナルED法が苦手としていた1つの重み空間での多クラス分類に対応しています。

### 3. Gabor特徴抽出

大脳皮質V1野の単純型細胞をモデル化したGaborフィルタによる特徴抽出を搭載しています（デフォルトON）。入力品質の向上により、適切なパラメータ設定であれば1層構成でも95%超の精度を達成します。

### 4. 生物学的に妥当な機能のみで学習

誤差関数の最小化や微分の連鎖律などの数学的な最適化理論に依存せず、生物学的に妥当な機能（アミン拡散、コラム構造、Gaborフィルタ）のみで学習を行います。

### 5. 学習が速い

学習データ数が十分に大きい場合には、1エポック目で最終テスト精度の90%以上の精度に到達します。多くのエポックを繰り返す必要がなく、少ないエポック数で高い精度が得られます。

また、1サンプルあたりの学習処理が高速です。同一条件（NumPy、CPU、サンプル個別学習）で比較した場合、誤差逆伝播法（BP法）と比べて**約3.7〜4.4倍高速**に動作します。逆伝播の計算が不要で更新対象のニューロンが少ないため、層数が増えるほど速度差が拡大します。

### 6. パラメータ調整が容易

パラメータ変更に対して安定的・単調的に反応し、誤差逆伝播法で見られるような勾配消失・勾配爆発による突然の学習崩壊が発生しません。パラメータが最適値から多少外れても精度が大きく下落するようなことが生じ難いため、ハイパーパラメータのチューニングが容易です。

### 7. リザバーコンピューティング的特性

各コラムに属するニューロン数を1個にした場合（`column_neurons=1`）には、リザバーコンピューティング方式での動作となります。隠れ層のコラムに属さないニューロンを固定ランダム重み（リザバー）として保持し、少数のコラムニューロンのみをオリジナルED法で学習させます。`column_neurons=10`（デフォルト: 2〜3層構成）では、各クラスに10個のコラムニューロンが割り当てられ、学習に参加するニューロンが増えることで各クラスをより多様な特徴で表現できるようになります。リザバーコンピューティング的な構造（大部分が固定重み）は維持されたまま、コラムニューロンの増加により分類性能が向上します。

### コラムED法とは

**コラムED法（Columnar Error Diffusion Method）** は、金子勇氏が考案したオリジナルED法のニューラルネットワークに、大脳皮質のコラム構造を導入した拡張実装です。

オリジナルED法は、脳の神経伝達物質の拡散をモデル化した生物学的に妥当な学習アルゴリズムですが、2クラス分類を前提として設計されていたため、10クラス以上の多クラス分類を行うことが困難でした。その原因は、ネットワーク内の隠れ層ニューロンがどの出力クラスに寄与するのかという情報を持っていなかったことにあります。

コラムED法では、大脳皮質に見られるコラム構造を導入し、隠れ層の**コラムニューロン**と**出力クラスニューロン**の対応関係を明確にしました。これにより、各コラムニューロンが特定のクラスの学習信号のみを受け取る仕組みが実現し、1つのネットワーク（重み空間）で多クラス分類が可能になりました。

さらに、大脳皮質V1野の単純型細胞をモデル化したGaborフィルタによる特徴抽出を組み合わせることで、学習メカニズム自体を変更することなく、テスト正解率が約6%向上しています（MNIST 1層: 90.37% → 96.13%）。

### オリジナルED法とは

**誤差拡散学習法（Error Diffusion Learning Algorithm: オリジナルED法）** は、金子勇氏が1999年に考案したニューラルネットワークの学習アルゴリズムです。

現在のニューラルネットワーク学習方法の主流である「微分の連鎖律を用いた誤差逆伝播法（BP法）」は、ネットワークの出力と正解データとの誤差を計算し、その結果を微分の連鎖律を用いて出力層から入力層に向かって逆方向に伝播させることで、各層のニューロンの重みとバイアスを効率的に更新するというものです。

しかし、オリジナルED法を考案された金子勇氏は、この生物学的に不自然な学習方法を受け入れることができず、生物学的に妥当な脳の神経伝達物質（ノルアドレナリン、ドーパミン等のアミン系物質）が空間に拡散する現象をモデル化し、オリジナルED法を開発されました。オリジナルED法では、アミン系物質の拡散情報に基づいて各層が**独立して**重みを更新します。

**BP法との本質的な違い:**

| | BP法（誤差逆伝播法） | オリジナルED法（誤差拡散学習法） |
|---|---|---|
| 誤差の伝播 | 微分の連鎖律で軸索を逆流 | アミン濃度として空間に拡散 |
| 各層の学習 | 後段の勾配に依存 | **各層が独立して学習** |
| 生物学的妥当性 | 低い | 高い |

> 詳細は [docs/ja/ED法_解説資料.md](docs/ja/ED法_解説資料.md) を参照してください。

### アミン拡散による重み更新

オリジナルED法の学習は、以下の手順で行われます：

```
1. 順伝播: 入力 → 隠れ層(tanh) → 出力層(SoftMax) → 予測
2. 誤差計算: 正解クラスの確率誤差 → アミン濃度に変換
3. アミン拡散: 出力層 → 隠れ層（コラム構造に沿って選択的に拡散）
4. 重み更新: アミン濃度 × 飽和抑制項 × 入力 で各層が独立に更新
```

**飽和抑制項**（オリジナルED法の核心）:

```
飽和抑制項 = |z| × (1 - |z|)
```

- これはシグモイド微分 `z(1-z)` でも tanh微分 `1-z²` でもありません
- ニューロンの活性値 `z` が飽和（±1に近い）するほど更新量が小さくなる仕組みです
- 微分の連鎖律を使わずに、各層が独立して適切な更新量を決定できます

### コラム構造

大脳皮質の第一次視覚野や連合野には、似た特性を持つニューロンが柱（コラム）状に集まった構造が見られます[*1]。本実装では、この構造を取り入れることでオリジナルED法の多クラス分類能力を拡張しています。

**コラム構造の動作:**

1. 隠れ層のニューロンをハニカム構造（2-3-3-2配置）で2次元空間に配置
2. 各クラスのコラム中心を空間上に配置（10クラスなら10個）
3. コラム中心に最も近いニューロンがそのクラスの**コラムニューロン**として割り当てられる
4. 学習時、正解クラスのコラムニューロンのみがアミン信号を受け取って重みを更新
5. コラムに属さないニューロンは固定ランダム重みのまま保持される（`column_neurons`の設定により学習ニューロン数は変わる）

![ハニカム構造の空間配置](images/hexagonal_column_structure_no_origin.png)

### Gabor特徴抽出

大脳皮質V1野（第一次視覚野）の単純型細胞は、特定の方位と周波数のエッジに選択的に反応します。本実装では、これをGaborフィルタでモデル化し、入力画像の特徴を抽出します（デフォルトON）。

**フィルタ構成:**
- 8方位 × 2周波数 = 16 Gaborフィルタ + 2 Sobelエッジフィルタ = **18フィルタ**
- カーネルサイズ: 11×11、プーリング: 4×4平均プーリング
- 出力次元: 784 → 882（MNIST 28×28の場合）

**効果:** Gabor特徴により、1層構成でもMNIST 90.37% → 96.13%（+5.76%）に向上します。学習メカニズム自体は変更せず、入力品質の向上によって精度を改善する生物学的に妥当なアプローチです。

### リザバーコンピューティング的特性

`column_neurons=1`（デフォルト: 1層構成）の設定では、本実装はリザバーコンピューティングと同等の動作原理で機能します。

**なぜcn=1でリザバーコンピューティングになるのか:**

`column_neurons=1`は各クラスに1個のコラムニューロンを割り当てる設定です。2048ニューロンの隠れ層で10クラス分類を行う場合、学習対象になるのはわずか10個（全体の0.5%）で、残り2038個のニューロンは初期化時のランダム重みが固定されたまま変化しません。

この構成は、リザバーコンピューティングの基本原理と正確に一致します：

| リザバーコンピューティング | コラムED法（cn=1） |
|---|---|
| **リザバー**: 固定ランダム結合で入力を高次元空間に射影 | **非コラムニューロン（2038個）**: 固定ランダム重みで入力を高次元空間に射影 |
| **読み出し層**: リザバーの出力から分類境界を学習 | **出力層**: 隠れ層全体の活性化パターンから分類を学習 |

固定ランダム重みのニューロン群は、入力データを高次元の非線形空間にランダム射影する役割を果たします。異なるランダム重みを持つ各ニューロンが入力の異なる側面に反応するため、元の入力空間では線形分離が困難なパターンでも、高次元射影後には分離しやすくなります。出力層はこの射影された表現から分類境界を学習するだけでよいため、少数のニューロンの学習だけで高い分類精度を達成できます。

この構成により、コラムED法は生物学的妥当性（少数のニューロンのみが学習）と高い汎化性能（固定重みによる過学習抑制）を両立しています。

**cn>1の場合:**

例えば`column_neurons=10`（デフォルト: 2〜3層構成）では、各クラスに10個のコラムニューロンが割り当てられます。2048ニューロンの隠れ層では100個（全体の約4.9%）が学習対象となり、残り1948個が固定ランダム重みを保持します。

cn=1と比較すると、学習に参加するニューロンが増えることで各クラスをより多様な特徴で表現できるようになります。リザバーコンピューティング的な構造（大部分が固定重み）は維持されたまま、コラムニューロンの増加により分類性能が向上します。

## パラメータ自動設定

メイン版では、隠れ層数に応じた最適パラメータが `config/hyperparameters.yaml` から自動的に読み込まれます。ユーザーが指定する必要があるのは、隠れ層構成・データサイズ・エポック数の最小限の引数のみです。

### 自動設定される主なパラメータ

| パラメータ | 1層 | 2層 | 3層 | 説明 |
|-----------|------|------|------|------|
| column_neurons | 1 | 10 | 10 | コラムニューロン数 |
| init_scales | [0.4, 1.0] | [0.7, 1.8, 0.8] | [0.7, 1.8, 1.8, 0.8] | 層別初期化スケール |
| output_lr | 0.15 | 0.15 | 0.15 | 出力層学習率 |
| non_column_lr | [0.15] | [0.15, 0.15] | [0.15, 0.15, 0.15] | 隠れ層基準学習率（層別）※ |
| column_lr_factors | [0.01] | [0.005, 0.003] | [0.005, 0.004, 0.002] | コラムニューロン学習率倍率（層別） |
| hidden_sparsity | 0.4 | [0.4, 0.4] | [0.4, 0.4, 0.4] | 隠れ層スパース率 |
| gradient_clip | 0.0001 | 0.0001 | 0.06 | 勾配クリッピング |

> ※ `non_column_lr` は隠れ層全体の基準学習率として使用されます。非コラムニューロンはアミン信号を受け取らないため実際には学習せず、学習するのはコラムニューロン（`column_lr_factors`の倍率で更新）と出力層（`output_lr`の学習率で更新）のみです。
>
> 4層以上のパラメータも `config/hyperparameters.yaml` に定義されています。

### カスタマイズ方法

1. **YAML直接編集**: `config/hyperparameters.yaml` をエディタで編集
2. **設定確認**: YAMLファイルを直接参照

> 万が一YAMLファイルを壊した場合は、`config/hyperparameters_initial.yaml` からコピーして復元できます。

## オリジナルED法への準拠

金子勇氏のCソースコードをリファレンスとし、オリジナルED法に準拠した実装であることを確認しています。

| 項目 | 実装 | 準拠 |
|------|------|------|
| 出力層飽和項 | `\|z\| × (1 - \|z\|)` | ✓ |
| 隠れ層飽和項 | `\|z\| × (1 - \|z\|)` | ✓ |
| アミン拡散 | コラム構造に沿った選択的拡散 | ✓ |
| 重み更新 | アミン濃度ベース、微分の連鎖律不使用 | ✓ |
| Dale's Principle | 興奮性・抑制性ニューロンペア | ✓ |
| SoftMax | 出力の確率化のみに使用（順伝播） | ✓ |
